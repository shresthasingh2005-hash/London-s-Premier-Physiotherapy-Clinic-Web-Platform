# Restore Physio — Premium Physiotherapy Clinic Website

> Professional, premium-grade website for a London-based physiotherapy clinic featuring a booking system, service pages, doctor profiles, testimonials, pricing plans, and a contact section.

## ✨ Features

- **Elegant Hero Section** — Deep teal gradient with "Move Better. Live Better." typography and clinic interior photo
- **Teal & Gold Theme** — Sophisticated medical aesthetic with earthy, trust-building tones
- **Doctor Profile** — "Meet Your Physiotherapist" section with Dr. James Mitchell's credentials (BSc, MCSP, HCPC)
- **Services Showcase** — Detailed service cards for different treatment types
- **How It Works** — Step-by-step patient journey from booking to recovery
- **Testimonials** — Patient reviews with star ratings and feedback
- **Pricing Plans** — Clear pricing tiers with feature comparison
- **Book Assessment CTA** — Prominent booking button in navigation and throughout the page
- **Contact Section** — Location, phone, email with working form
- **Smooth Scroll Navigation** — Single-page navigation with smooth scrolling
- **AI-Generated Imagery** — Professional clinic and doctor portraits
- **Fully Responsive** — Adapts seamlessly from desktop to mobile
